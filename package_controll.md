# Package Controll

## 安装
- Python code
打开`https://packagecontrol.io`
切换至`installation`
`copy python code` -> 复制
ctrl + ` -> 打开Sublime控制台
`paste python code` -> 粘贴

- Command Palette
`ctrl + shift + p` -> 打开命令面板
`Install Package Control` -> 输入这个命令

- github
download `Package Control.sublime-package`
