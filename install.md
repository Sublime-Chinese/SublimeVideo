## 卸载
- Mac
    - 安装目录: `/Applications/Sublime Text.app/Contents/MacOS`
    - 本地Packages目录: `~/Library/Application Support/Sublime Text 3`
- Windows

## 还原到初始状态
- 删除本地Packages目录

## 6个小技巧
- 通过多选, 批量修改变量名字
> Ctrl + d
    - 全选单词
    - 点选单词
- 多行编辑
    - ⇧⌘L
    - ctrl + j
- 命令面板
    - ⇧⌘P
    - ctrl + shift + p
- 通过正则表达式查找, 替换
- Goto Anything
